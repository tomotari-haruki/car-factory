# 近畿自動車株式会社様 提案用デモ

このディレクトリは、近畿自動車株式会社様向けのホームページ改善提案デモです。  
公式サイトではなく、営業提案時に改善イメージを見せるための非公式デモです。

## 見方

`prototype/index.html` をブラウザで開いて確認できます。  
ローカルサーバーで見る場合は、プロジェクトルートで以下を実行します。

```bash
python3 -m http.server 8080 -d prototype
```

その後、`http://localhost:8080/` を開きます。

## 構成

- `index.html`: デモページ本体
- `assets/styles.css`: レスポンシブCSS
- `assets/app.js`: メニュー開閉、ヘッダー状態切り替え、FAQ、フォーム風導線の確認表示
- `assets/hero-workshop.webp`: 提案用に生成した汎用整備工場イメージ
- `assets/favicon.svg`: 提案用の汎用ファビコン

外部ライブラリは使用していません。PHP化する場合は、見積もりフォーム風エリアを既存問い合わせ処理へ接続しやすい構成にしています。

## 公開時の注意点

- `meta name="robots" content="noindex,nofollow,noarchive,nosnippet,noimageindex"` を設定済みです。
- `meta name="googlebot" content="noindex,nofollow,noarchive,nosnippet,noimageindex"` を設定済みです。
- ページ内に「これは提案用の非公式デモです。近畿自動車株式会社様の公式サイトではありません。」と明記しています。
- 公式写真、公式ロゴ、架空の口コミ、架空の実績、架空のキャンペーンは使用していません。
- 電話番号、住所、営業時間、定休日、LINE URLは公式サイトで確認できた情報を使用しています。
- 営業時間は `8:30-17:30`、定休日は `第二第四土日、祝日` として掲載しています。
- 最新の休業日は公式サイトまたはお電話で確認する注記を入れています。
- デモサイト本体には提案者側の制作費を載せません。
- 制作費や保守費は `docs/proposal.md` / `docs/sales-one-pager.md` で説明します。
- デモURLを送る際は、本文で「非公式デモ」であることを明記してください。
- 代車、即日対応、Google評価、口コミ数など、確認できていない情報は掲載していません。
- GitHub Pagesなどで一時公開する場合も、URLを営業先以外へ広く拡散しない運用を推奨します。

## Lighthouse想定の改善点

本デモは軽量なHTML/CSS/JSで作っていますが、本番化前には以下を確認してください。

- hero画像は `preload` と `fetchpriority="high"` を設定済みです。本番では実写真に合わせて再圧縮してください。
- WebP以外のフォールバック画像が必要か
- 実運用フォーム接続後のJavaScript量
- 実店舗写真を使う場合の画像圧縮と遅延読み込み
- OGP画像の絶対URL設定
- 本番公開時のnoindex解除タイミング

## アクセシビリティの確認点

- 375px幅で横スクロールが出ないことを確認する。
- 固定下部CTAが本文やフッターの重要情報を隠しすぎないことを確認する。
- 実写真を入れる場合は、意味のある画像に適切なaltを入れる。
- 料金表やFAQは、色だけで違いを伝えない。
- フォーム接続時は、エラー文と必須表示をスクリーンリーダーでも分かる形にする。

## 実施済みチェック

- `html-validate` でHTMLバリデーション通過。
- `axe-core` 自動チェックで0 violationsを確認。
- 375px / 768px / 1280px で横スクロールが出ないことを確認。
- 固定下部CTAは375pxで表示、768px以上では通常導線に切り替わることを確認。
- ローカルアンカー切れ、ローカルアセット欠落がないことを確認。
