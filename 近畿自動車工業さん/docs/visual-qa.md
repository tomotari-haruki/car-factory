# デモサイト最終UIチェックメモ

## 目的

`prototype/index.html` は、近畿自動車株式会社様のリニューアル後サイトを想像していただくための非公式デモです。営業資料ではなく、エンドユーザー向けサイトとして自然に見えることを優先しています。

## 今回のUI調整方針

- 白、濃紺、整備工場らしいオレンジ、安心感のあるグリーンで、地域企業らしい落ち着いた印象に整理。
- ヒーロー画像、カード、CTA、フォーム、FAQ、地図風パネルに奥行きと反応を追加。
- スマホでは電話、LINE、見積もりの3つだけを下部固定CTAにして、押しやすさを優先。
- 制作費や保守費など提案者側の情報はデモ本体に載せず、`docs/proposal.md` と `docs/sales-one-pager.md` に分離。
- アニメーションは控えめにし、本文を隠さない形で実装。

## 営業先に見せる前の見どころ

- ファーストビューで「指定工場」「国家資格整備士」「車検見積もり」「電話」「LINE」がすぐ分かる。
- 改善ポイントとBefore / Afterで、今ある強みを否定せず整理する提案だと伝わる。
- 料金表はスマホでカード表示になり、比較しやすい。
- 相談内容は車検、修理、板金、法人・商用車に分かれており、来店前の迷いを減らす。
- アクセス欄にはGoogleマップへのリンクを設定済み。

## 確認済み

- `prototype/index.html` に制作費の「5万円プラン」セクションは入れていない。
- 見積もりフォーム風CTAに「ホームページ改善について相談」は入れていない。
- `meta name="robots"` と `meta name="googlebot"` に `noindex,nofollow,noarchive,nosnippet,noimageindex` を設定。
- 非公式デモ表記をヘッダーとフッターに表示。
- 営業時間は `8:30-17:30`、定休日は `第二第四土日、祝日`。
- JSON-LDの営業時間も `08:30` から `17:30`。
- 375px / 768px / 1280px で横スクロールなし。
- `axe-core` 自動チェックで0 violations。

## 本番化前に必ず確認すること

- 公式写真、ロゴ、スタッフ写真の使用許可。
- 車検料金表の最新性。
- 休業日カレンダーと月ごとの休業日変更。
- Googleビジネスプロフィールの営業時間、評価、口コミ数。
- 代車、即日対応、出張引取、納車対応の可否。
- 問い合わせフォームの送信先と個人情報保護方針。
